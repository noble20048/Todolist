# To-Do List App

A simple to-do list app built with HTML, CSS, and JavaScript.

## About

<img width="1366" height="768" alt="image" src="https://github.com/user-attachments/assets/c5fa836f-0e8e-4057-a7c1-fb2fb2b00450" />

This project was built by following a GreatStack tutorial on YouTube. I typed along with the instructor to build the app structure, styling, and functionality. The tutorial moved fast and I wasn't able to fully grasp every concept while building it — this was primarily a typing-along exercise to get familiar with how a to-do app is structured (DOM manipulation, event listeners, local storage) rather than a fully self-understood build.

A few bugs were fixed afterward using AI assistance, and the UI was tweaked to move away from the default tutorial look.

## Tech Stack

- HTML
- CSS
- JavaScript (DOM manipulation, event handling)

## Features

- Add new tasks
- Mark tasks as complete
- Delete tasks
- Basic responsive UI

## Status

Functional. Built primarily as a learning/practice project to get comfortable with JavaScript syntax and DOM concepts — revisiting parts of this later to actually understand the underlying logic is a planned next step.

## How to Run

1. Clone or download this repository
2. Open `index.html` in your browser

## Learning Note

This was a "type-along" project. I'm not claiming deep understanding of every function here — it's part of my ongoing process of learning JavaScript by building and revisiting code.
