# TaskFlow — To-Do List App

<img width="1366" height="768" alt="image" src="https://github.com/user-attachments/assets/c5fa836f-0e8e-4057-a7c1-fb2fb2b00450" />



A clean, responsive to-do list app built with vanilla HTML, CSS, and JavaScript.

## Features
- Add and delete tasks
- Mark tasks as complete
- Responsive UI
- Local storage persistence

## Tech Stack
HTML, CSS, JavaScript (DOM manipulation, event listeners, local storage)

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
